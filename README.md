# 🦚 Dwapara Yuga

Experience the Divine Journey of Shri Krishna.

## 🎮 About

Dwapara Yuga is an educational adventure game inspired by the life and teachings of Shri Krishna. Players will explore iconic locations, complete quests, solve puzzles, answer quizzes, and experience important events from Krishna's life through interactive gameplay.

## ✨ Planned Features

- 📖 Story Mode
- 🏆 Global Leaderboard
- 🎁 Daily Rewards
- 👤 Player Profile
- ☁️ Cloud Save
- 🌍 Multiple Languages
- 🎵 Indian-inspired Music
- 🎯 Quests & Achievements
